
# Installation

```bash
git clone https://github.com/rushikesh456thorat/BookingBuddy.git
cd BookingBuddy
npm install
```
# How To Run

```bash

npm run dev
```
Click on link of localhost in terminal

# How to push repo

```bash
git init
git add .
git commit -m "enter your commit message"
git pull origin main
git push -u origin master
```
